## 🔍 Task 1: What Is the Purpose of a Firewall?
חומת אש היא רכיב אבטחה המנטר ומסנן תעבורת רשת נכנסת ויוצאת.
* **האנלוגיה:** מאבטח בכניסה לבניין הבודק מי מורשה להיכנס.
* **תפקיד ה-PT:** הבנת חומת האש חיונית כדי לזהות חוקים שניתן לעקוף או לנצל (Firewall Evasion).

**שאלה ותשובה:**
* **Q:** איזה פתרון אבטחה בודק תעבורה נכנסת ויוצאת?
* **A:** `Firewall`

## 🛠️ Task 2: Types of Firewalls
קיימים סוגים שונים של חומות אש הפועלים בשכבות שונות של מודל ה-OSI:

| סוג Firewall | שכבת OSI | תיאור |
| :--- | :--- | :--- |
| **Stateless** | 3, 4 | בודק כל חבילת מידע בנפרד ללא קשר לחיבורים קודמים. מהיר אך פחות מאובטח. |
| **Stateful** | 3, 4 | עוקב אחר מצב החיבורים (State Table). זוכר אם חבילה היא חלק משיחה מאושרת. |
| **Proxy (Application)** | 7 | משמש כמתווך. בודק את תוכן המידע (Payload) בשכבת האפליקציה. |
| **Next-Gen (NGFW)** | 3-7 | הפתרון המתקדם ביותר. כולל IPS, פענוח SSL/TLS וניתוח התנהגותי (Heuristic). |

**שאלות ותשובות:**
1.  **Q:** איזה סוג שומר על מצב החיבורים? **A:** `Stateful Firewall`
2.  **Q:** איזה סוג מציע ניתוח היוריסטי? **A:** `Next-Generation Firewall`
3.  **Q:** איזה סוג בודק תעבורה המיועדת לאפליקציה? **A:** `Proxy Firewall`

## 📜 Task 3: Rules in Firewalls
חוקים (Rules) הם ה"מוח" של חומת האש. כל חוק מורכב מ:
* **כתובות (Source/Destination):** מאיפה ולאן.
* **פורטים ופרוטוקולים:** (למשל TCP 80, UDP 53).
* **פעולה (Action):** Allow (אפשר), Deny (חסום), Forward (העבר).
* **כיוון (Direction):** Inbound (נכנס) או Outbound (יוצא).

**שאלות ותשובות:**
1.  **Q:** איזו פעולה מאפשרת תעבורה? **A:** `Allow`
2.  **Q:** מה הכיוון של תעבורה שעוזבת את הרשת שלנו? **A:** `Outbound`

## 💻 Task 4: Hands-on Windows Defender Firewall
בשלב זה התחברנו למכונת Windows מרוחקת כדי לחקור חוקים קיימים וללמוד כיצד להגדיר חוקים מותאמים אישית (Custom Rules).

### תהליך העבודה במעבדה:
1.  גישה ל-**Advanced Settings** ב-Windows Defender.
2.  זיהוי חוקי ה-**Inbound** שהוגדרו עבור שירות ה-SSH (פורט 22).
3.  בדיקת ה-**Scope** של החוקים כדי לראות הגבלות לפי כתובות IP.

### Exercise - ניתוח הממצאים:
* **שם החוק שחוסם SSH נכנס:** `Block SSH`
* **שם החוק שמאפשר SSH מכתובת ספציפית:** `Allow SSH`
* **כתובת ה-IP המורשית בחוק הנ"ל:** `192.168.1.10`
